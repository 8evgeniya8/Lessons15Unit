Задача: Реверс та Аналіз Рядка

Створити клас, який буде мати наступні методи:

1) Реверс рядка: приймає рядок як параметр та повертає його у зворотному порядку.

2) Перевірка на паліндром: приймає рядок та повертає true, якщо рядок
є паліндромом (читається однаково в обох напрямках), та false в іншому випадку.

3) Підрахунок слів: приймає рядок та повертає кількість слів у ньому.
